Feature Icons

Feature icons are colored, styled icons representing each feature. You can use the .feature-icon class in your scss or html to acheive this effect.

Features

Features that take advatage of the feature layout classes should be wrapped by an element with the class kp-feature

Feature Layout

The feature menu represents the main links associated with a feature. On mobile, it will be visible on the landing page, and hidden on every other screen. On desktop, it will be represented as a side panel of navigation.

Activating the Landing Page on Mobile

To designate that the current page is the landing page, add the landing-page class to a parent element. This will hide the feature content, and use the .feature-header, .feature-description, and .feature-menu elements to form the landing page.
