Fancy Select

You can implement a fancy-select by adding proper import and then calling with the id of select element.

Note: div and class aqua are just for display purposes, you would not need to add these inline styles to your code.

Example is showing with a default selected option.

label is used to prefix the seletion.

The first option is selected by default, Apple in this case. Long option text will be wrapped when it's needed.

              import fancySelect from 'styleguide/lib/modern/scripts/fancy_select';

              fancySelect($("#select-from"));

Example when there is only one option.

Example is showing with the placeholder: Select from.

If you want to specify a placeholder, set title="Select from"on the label

NOTE: the placeholder is over-ridden if you specify one of the options as selected="true".
