Fancy Select

You can implement a fancy-select by adding proper import and then calling with the id of select element.

Note: div and class aqua are just for display purposes, you would not need to add these inline styles to your code.

Example showing with a default selected option Apple.

   
              import fancySelect from 'styleguide/lib/modern/scripts/fancy_select';

              fancySelect($("#select-from"));

Example showing when there is only one option

Example showing with the description Select from only for display:

By adding value="default-option-for-display" to the option, you could have default description set but not appear in the list of options when the dropdown is open.

When you only need to show one option in the style above, you could use -item.
