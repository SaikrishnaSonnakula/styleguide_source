Grid Layout

To use your own class names and have a custom grid setup, use the @include span-columns(#); in your scss. What this include does is specifies the number of columns an element should span. If the selector is nested the number of columns of its parent element should be passed as an argument as well. Also You do not have to use the container class, Just use the include the @include outer-container to any parent container class.

If you prefer to use the class based system, please make sure to wrap your .column-# class in .row then a parent .container

For more examples of this please refer to the bourbon/Neat grid

Using the bourbon includes


        .custom-main-body-container-name {
          @include grid__container;
        }
        .custom-two-column-layout-name {
           @include grid__column-(6);
          }
