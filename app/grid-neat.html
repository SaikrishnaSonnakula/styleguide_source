Bourbon Grid Layout

To use the 2 column layout, use the @include span-columns(6); within your parent class. You do not have to use the container class. Just use the include the @include outer-container to any parent container class

          @include outer-container;
        }
        .custom-row-for-columns-name {
          @include row();
        }
        .custom-two-column-columns-name {
         @include span-columns(2);
