Buttons

<a> elements, <button> elements can be styled like a button. Note: These two elements have different semantic meaning; choose the one most appropriate for you. Rule of thumb: a link takes you to another place, a button does something.

For an inverted color, use the -inverted class.

Append the class name "kp-radio-btns" to apend the styles of to your project radio buttons. Use the attribute checked="checked" to preselect a radio button.


Fieldset and legend should only be used to associate groups of controls when a higher level description is needed. Do not use nested fieldsets. Single checkboxes or basic radio buttons that make sense from their labels alone do not require fieldset or legend.
