About

The Web Design Styleguide (or as we call it, the styleguide) is both a styleguide document, which you can see here on this site, and a library of styling assets, like images, fonts, and stylesheets. This document was built with the same assets that are imported into responsive KP applications.

If your interested in learning more about this concept, listen to this talk on Styleguide Driven Development by Nicole Sullivan.

Installation

The styleguide is NPM module. You can install it by running the following command in your project directory: 

npm install -D git+ssh://git@stash.kp.org/rwd/styleguide.git

Then, include the styles your sass file: 

@import "styleguide/modern"

Since the styleguide contains a variety of front-end assets, which all need to be handled the right way, we encourage you to use the KP Build Plugin. If you are using the Build Plugin, you're all set!

If for some reason you can not use the Build Plugin, you will need to build the assets manually. We will soon be documenting this process, but in the mean time, please contact shane.russell@kp.org or tommy.wu@kp.org for assistance.
