Using the Table Sass Mixin

Use the mixin @extend %table-base to apply your own unique styling to the table within your class that usess the double hyphen after the .table class name. In the example below the custom class name --grey-even-row is appended to the defualt .table class


          .table--grey-even-row {
            @extend %table-base;
            tr:nth-of-type(even) {
              background: color("graphite-light");
            }
          }
